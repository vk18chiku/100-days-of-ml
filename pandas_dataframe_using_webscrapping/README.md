# Pandas DataFrame Using Web Scraping

This folder contains notebooks for web scraping and creating DataFrames.

## Contents
- `webscraping.ipynb` - Web scraping techniques and DataFrame creation
- `books_data.csv` - Sample scraped data

## Learning Topics
- Web scraping with BeautifulSoup
- Parsing HTML/XML
- Extracting structured data
- Converting scraped data to DataFrame
- Data cleaning from web sources

## Key Skills
- HTTP requests for web pages
- HTML parsing and navigation
- CSS selectors and XPath
- Data extraction and transformation
- Handling web scraping challenges

## Important Notes
- Always check website's robots.txt and terms of service
- Use appropriate delays between requests
- Respect server resources
