# 100 Days of Machine Learning

A comprehensive learning journey through core machine learning and data science concepts. This project documents daily progress on various ML fundamentals and applications.

## Project Structure

### 1. **API to DataFrame**
- Converting API responses to pandas DataFrames
- Working with REST APIs and JSON data
- Data integration from web services

### 2. **Bivariate Analysis**
- Relationship analysis between two variables
- Correlation and covariance
- Advanced visualization techniques (heatmaps, pair plots)

### 3. **Pandas Profiling**
- Automated exploratory data analysis
- Comprehensive data quality reports
- Distribution and correlation analysis

### 4. **Pandas DataFrame Using Web Scraping**
- Web scraping with BeautifulSoup
- HTML parsing and data extraction
- Converting unstructured web data to DataFrames

### 5. **Standardization/Feature Scaling**
- StandardScaler and normalization techniques
- Impact on different ML algorithms
- Comparing Logistic Regression vs Decision Tree

### 6. **Understanding Your Data**
- Initial data exploration
- Descriptive statistics
- Data profiling and quality assessment

### 7. **Univariate Analysis**
- Single variable analysis
- Distribution visualization
- Outlier detection and handling

### 8. **Working with CSV Files**
- Reading and writing CSV data
- Handling different encodings and delimiters
- Large file processing

### 9. **Working with JSON**
- JSON parsing and manipulation
- Nested data handling
- Creating DataFrames from JSON

## Technologies & Libraries
- **Python 3.x**
- **Pandas** - Data manipulation
- **NumPy** - Numerical computing
- **Matplotlib & Seaborn** - Visualization
- **Scikit-learn** - Machine Learning
- **BeautifulSoup** - Web scraping
- **pandas-profiling** - Automated EDA

## Key Learning Outcomes
✓ Data loading and exploration
✓ Data cleaning and preprocessing
✓ Feature scaling and normalization
✓ Exploratory data analysis techniques
✓ Machine learning model building
✓ Performance evaluation metrics

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/vk18chiku/100-days-of-ml.git
cd 100-days-of-ml
```

2. Install dependencies:
```bash
pip install pandas numpy matplotlib seaborn scikit-learn beautifulsoup4 pandas-profiling
```

3. Explore individual project folders with their dedicated README files

## How to Use
Each folder contains:
- Jupyter notebooks with explanations
- Sample datasets
- README with specific learning objectives
- Code examples and visualizations
