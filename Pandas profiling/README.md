# Pandas Profiling

This folder demonstrates automated data profiling using pandas-profiling.

## Contents
- `profiling.ipynb` - Automated data profiling notebook
- `output.html` - Generated HTML report with comprehensive data profile
- `zomato.csv` - Sample dataset for profiling

## Learning Topics
- Automated exploratory data analysis (EDA)
- Data quality assessment
- Missing value analysis
- Distribution analysis
- Correlation detection
- HTML report generation

## Key Skills
- Using pandas-profiling library
- Generating comprehensive data summaries
- Understanding data characteristics
- Identifying data issues automatically

## How to View the Report
Open `output.html` in a web browser to view the detailed profiling report.
