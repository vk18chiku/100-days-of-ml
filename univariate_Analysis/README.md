# Univariate Analysis

This folder contains notebooks for analyzing single variables.

## Contents
- `analysis.ipynb` - Univariate analysis techniques
- `Titanic-Dataset.csv` - Sample dataset for analysis

## Learning Topics
- Distribution analysis
- Histogram and density plots
- Categorical variables analysis
- Count plots and bar charts
- Statistical measures (mean, median, mode, std, etc.)
- Skewness and kurtosis
- Outlier detection

## Key Skills
- Using matplotlib and seaborn for visualization
- Understanding data distributions
- Identifying skewed distributions
- Detecting outliers and anomalies
- Visualizing single variable summaries

## Visualization Types
- Histograms
- Density plots (KDE)
- Box plots
- Count plots
- Violin plots
