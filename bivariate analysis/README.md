# Bivariate Analysis

This folder contains notebooks for analyzing relationships between two variables.

## Contents
- `bivariate_analysis.ipynb` - Techniques for analyzing two variables simultaneously
- `Titanic-Dataset.csv` - Sample dataset for analysis

## Learning Topics
- Scatter plots and correlation analysis
- Categorical vs Numerical relationships
- Box plots and violin plots
- Heatmaps and crosstabs
- Statistical measures (correlation, covariance)

## Key Skills
- Using seaborn for bivariate visualizations
- Understanding relationships between variables
- Categorical and numerical variable analysis
