# Understanding Your Data

This folder contains notebooks for initial data exploration and understanding.

## Contents
- `understanding.ipynb` - Exploratory data analysis techniques
- `netflix_titles.csv` - Sample dataset for analysis

## Learning Topics
- Data loading and inspection
- Shape, size, and structure of data
- Data types and continuous vs categorical variables
- Basic statistics and summary
- Missing values detection
- Data overview and head/tail exploration

## Key Skills
- Using pandas for data inspection
- Descriptive statistics
- Data profiling basics
- Identifying data quality issues
- Initial data exploration techniques

## Questions to Ask
- What does the data contain?
- What are the dimensions?
- What are the data types?
- Are there missing values?
- What patterns exist?
